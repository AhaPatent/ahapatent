> 任何天赋都可以产生收入。关键是要有价值，找到那个能把自己天赋发挥出价值的那个位置，就是呼召，或者，天职。

```
所以，连接点、价值点、呼召，是同一个概念。
价值点和人脉管理是相辅相成的关系。找到了自己的价值点，人脉管理会更有效。 反过来，人脉圈能给你第二次机会，帮你一次又一次校准方向，找到你一生的呼召。
找到这个连接点以后，我们来讲怎样用有限的时间精力，用有效的方法经营人脉，释放出朋友圈的真正力量。
```

在圈子和圈子之间，有空隙，这就是社会学家说的 “结构洞” 。串联者就是在圈子之间做连接，填补结构洞的那个人。 你的社会资本一定程度上，以你个人弥合了多少空洞来衡量。

产品经理、项目经理、高管助理的工作有个共同特点： 需要多人合作才能完成任务，而他们却没有针对那些合作者的正式授权和奖惩机制。 所以，他们如果要向前推进工作，靠的是真正的领导力，也就是非职权领导力。

> 这种领导力不是职位赋予的，而是一种人际关系上的影响力。当项目卡在这里动不了时，他们有本领将项目lead往前走，这就是真正的leadership。

有什么方法呢？ 你可以让自己成为信息的周转站。

《金牌律师》的秘书Donna是我们学习的榜样。

相比于那些无所不能的律师，Donna掌握的信息最多，最能帮到大家。Donna和每个人交谈，每个人都愿意和她聊，她在大多数人之前获得重要信息，甚至比当事人还早获得。她的人际影响力来自于她的信息，所以，她能将问题解决，将事情向前推进。在这个方面，她配得上她的高薪。

怎样让自己成为信息周转站呢？ 你可以根据物理位置来构筑你在人际网络的中心位置，让你的办公桌处于交通枢纽、宇宙中心，中心位置会带来权力。

更重要的是， 你要有亲和力， 让周围的人愿意靠近你，和你交流。怎样和人聊天，你可以参照我的上一门课程中的闲谈技巧。

在闲谈中，多多挖掘情报。 参考领英创始人Reid Hoffman的建议，你在和跨部门、跨行业的人交往时，可以用这些问题来挖掘情报：

```
决定我们行业发展方向的关键技术趋势是什么？（比如大数据）
我们的竞争者在做些什么有用的或没用的工作？
我们应该和哪些关键的业内人士打交道？
我们行业的人才招聘新趋势是什么？
```

> 你手里的情报越多，个人威望度就越高，也越容易将问题解决，将项目推进。

如果你在大公司工作，更要学会利用公司资源来建立人脉。在大公司里，千万不要将自己角色化、工具化。 这意味着，你在公司外建立的人际关系，可能重要过你在公司内建立的人际关系。

> 如今这个时代，没有什么东西能为我们的事业保驾护航。而在你无路可走的时候，人脉具备产生瞬间作用的能量。

若真心想组织一次聚会，还怕找不到理由？你的理由可以是：`周末放松、庆祝国庆、品尝你收藏的红酒、品尝母亲从老家带来的特产、一起欣赏你们旅游回来后制作的小影片，等等。`

### 7. 更新人脉的三个原则

![image](https://user-images.githubusercontent.com/99159300/152738291-2663d42e-913e-48ed-9ae6-cd0e29496070.png)

遇到损友分三步走

接下来我告诉你第二个原则，遇到损友怎么办？

损友比“耗能”型朋友更糟糕，他们做出了一些对不起你的事情，比如诽谤你、取乐你。面对这样的损友，你该怎么办呢？

我给出三个步骤的建议：

第一步， 你问问自己，是不是已经对他们表达了你的友善，如果你之前一直忽略了他们，那么，在他们的世界里，你就不是朋友，他们做些不道德的事情，也更容易被宽恕。

第二步， 如果你之前确实从未对他们表达过友善，我建议你补上这个动作，因为你想让别人怎样对待你，你就应该怎样对待他人。 

你后补上的这个友善，给了双方一个机会不去树敌，而是交朋友。 

人生不易，能少一个敌人就尽量少一个。

那话说回来，如果你之前已经对他们表达过友善了，他们却没领情，对你不友善，那么，直接忽略他，如果他们实实在在伤害到了你的利益，你就反击。

第三步， 如果一段时间之后，他转变了态度，想过来和你交朋友，那就尽释前嫌。所谓不打不相识，面对未来永远比纠结于过去更有格局。

这就是遇到损友的三个步骤。

#### 人脉归档表和155人黄金人脉圈样表，帮你清晰地盘点已有人脉
工具1：人脉归档表
用途：盘点已有人脉、找出155人黄金人脉圈

盘点已有人脉-样表：
![image](https://user-images.githubusercontent.com/99159300/152739176-421fdd27-f53c-468b-b1e7-90a9a33b696b.png)
这份人脉归档表包括以下几栏：名字、角色、职业、地区、行业、影响力、和你的亲密度。其中影响力指的是他在他那个领域内的影响力，分为弱、中、强；和你的亲密度也分为疏、中、密。

你做好这个表后，就要开始做这四项观察。

```
首先，观察“角色”一栏， 你看看，自己的交友模式多集中在工作上，还是私下生活中。
其次，观察“地区”一栏， 你看看地区是不是足够多元化。
然后，观察“行业”一栏， 你检查一下是不是六大生态圈的联系人都有。这六大生态圈是：
与兴趣爱好相关的联系人：在俱乐部里建立的友谊，比职场上建立的更真实。
与工作相关的联系人：包括客户、供应商、竞争对手等等。
政府：这个生态圈可以帮你更深入、更生动地了解国家的宏观政策。
金融：这个生态圈当然重要，钱都在他们那儿。
媒体：这个生态圈对你打品牌、应对公关危机有重要价值。
社区：这个生态圈让你接地气。
```

最后，你再观察一下，在这个人脉归档表里，你有没有找回几个温暖的名字？如果有，你要马上和他们重新取得联系。

这份表格可能还会改变你的沟通习惯，当你遇到问题的时候，你会更有针对性地更有方向地去主动挖掘已有资源。

![image](https://user-images.githubusercontent.com/99159300/152739192-be90d29d-d8a6-4a24-b8ab-501654f96fb0.png)

针对你做出来的人脉归档表， 最后那两栏就能帮你圈定自己的155人黄金人脉。包括命友5人、密友50人，好友100人。
只要联络人的这两栏组合不是“弱”+“疏”，那他就应该进入你的黄金人脉圈。
也就是说，只要这个人在自己行业里有中等或中等以上的影响力，或者，他和你的亲密度达到中等或中等以上，那就应该进入你的155人的黄金人脉圈里。

#### 初次见面前的邮件/信息模板，为你们的初次见面造势，给对方施加影响

工具4：初次见面前的邮件/信息模板
用途：为你们的初次见面造势，给对方施加影响。

模板：

文字要依次包括这三个内容——

你是谁；
你仰慕他；
你能给他带来价值。

第二项内容“你仰慕他”，要写具体。 比如，你可以咀嚼一下他说过的某句话，说说这句话给你带来的影响，还可以为这句话赋予新的含义。

> 变抱怨为感恩，变抵抗为接受，变判断为观察，变恐惧为发现

[命友、密友和好友各自的联系频率](https://dedao.cn/course/article?id=D4vE8rn63yN5JApWOVOpoPzG0MdqBb)

命友是我们不可取代的财富。

对于这5个人，你要有事没事都联系一下，内容不重要，而联系他的这个动作更重要。 在他们身上的投入，我们不期待事业上的回报，而是情感回报，而且一定会有加倍的情感回报。

我们容易在命友5人身上犯的错误是，很少向他们求助。 我们以为他们理所当然地知道我们在做什么，我们还臆想他们可能是帮不上忙的。其实，你有了困难，应该第一时间告诉他们，他们会用自己积累了一辈子的资源，全力相助。

对于密友50人，你们的联系频率至少每周一次， 比如微信点赞，微博私信，喝喝咖啡，约着一起打羽毛球，等等。

对于好友100人，你至少每个月要联络一次。 节日、生日、与他相关的重大事件发生时，一定要和他联系。

有人问：“我自己觉得命友密友也不适合天天联络或者过于频繁联络，特别是见面的交流，会在一定程度上打乱他们的生活节奏吧。但是到底怎么样的一个时间周期联系比较好呢？”

除了是亲密的生活伴侣，谁都受不了天天联络。对于命友，我不做硬性规定，有事没事都可以联络。如果双方都有兴致，可以多聊一会儿。如果没有兴致，打个招呼就行。命友嘛，关系已经很铁了，可以随性一些。

密友呢？我建议联系频率至少每周一次。朋友圈里点个赞或留个评论，是一种最不会打扰到对方的联络方式。让他有一种你一直都在的感觉。

总之，通往幸福的道路，不只是你找到这155个会让你幸福的人，其实反过来说才是真理：通往幸福的道路，是你找到你想让他幸福的那155个人。你用规律性的联系频率，将自己的时间和才华给予他们，你们从亲密友爱的关系中得到深层次的快乐。

有人问：“如何与过去曾经有过节或有误会的朋友重新建立新的良好关系，因为出现这些问题后往往会沟通有隔阂。”

我觉得坦诚地谈一次是个不错的选择。这也是罗老师在《知识就是力量》节目里，如何重获信任中提到的，行动。用行动表示你在主动建设这段关系。

谈什么呢？就谈上次的过节和误会。说说你的感觉，澄清你的动机，表达你的态度。

我在工作中，如果犯了错误，或者领导误会我了，我会鼓起勇气去和他谈。

建立关系，一定是双向的。你对他已经有足够的兴趣，那么，这个问题实际问的是，怎样让他对你也有足够的兴趣。

成为一个有魅力的人，别人就会对你产生兴趣。 魅力这个话题，很有趣，也富于争议。我本来觉得魅力是天赋，后来我发现，魅力原来也可以学得会。

![image](https://user-images.githubusercontent.com/99159300/152740894-cada78e1-29b0-4625-8043-378d1cd01b90.png)
> 1955年，一位摄影师跟着玛丽莲·梦露拍照，在地铁里，她是个路人，她就是Norma Baker，这是她的原名。没有人认出她，到了热闹的纽约街道，梦露环顾一下四周，打趣地问摄影师，你想不想看到梦露？接着，摆弄了一下头发，换了个姿势，她身上的魅力就像开关一样打开了，人们发现大明星就在旁边，把她团团围住。

创造魅力的公式=权威性+亲和力+专注当下。

魅力的法力无边，魅力是后天修炼所得。

第一个，权威性。


他比你年长，级别又高，在这样的人面前要展示权威性，比较难。不过，你可以借力于：你说的这个信息来自一手资料、亲身体会、专家的意见、书中的智慧，或团队的一致看法，从这些角度来提供信息，增加自己的权威性。

并且，要建立自己的权威性，还有三个很有意思的小细节你可以模仿：

在一段话结尾处，降低声调；
减少鸡啄米一样的点头次数；
说话前，停顿两秒。

第二个，亲和力。

成为一个有趣的人，不遮不掩，不端也不装。 除了在你和他见面时，传递这个形象，你还可以通过朋友圈。检查一下你的朋友圈信息，你发的朋友圈，是不是一个有血有肉，有爱有恨的真实形象。

如果他根本不了解你，也无从喜欢你。给机会让他了解你。就像美国现代小说家约翰·斯坦贝克说的：“如果你们相互了解，就会彼此仁慈。了解一个人永远不会导致仇恨，它几乎总是会带来爱。”

第三个，专注当下。

这个容易做到。你专注于当下的程度，能让你从人群中脱颖而出。

你看，魅力和年龄、阅历，没什么关系。一个年龄大、阅历多的人，如果没有魅力，反而会让人觉得很阴森。所以，作为年轻人，应该完全有信心把自己身上的魅力潜质引导出来，用自己的魅力去交朋友。

有人问：“请问老师，我应该怎样与自己不喜欢却又一定要接触的人相处。”

如果我们有一根魔法棒，挥一挥就能改变对方，多好。可惜没有，我们连一根能让自己一挥而变的魔法棒都没有。

首先，我们要意识到，真正愚蠢狂妄的人，有，但比例其实很小。每当我遇到很难相处的人的时候，我会用这两句话来提醒自己。

第一句话： 在他的世界里，他真的努力了，你相信吗？如果你真的相信。沟通中的不愉快，会减少一半。

第二句话： 我们永远不可能有足够的信息去给一个人下论断。

如果你相信这句话，沟通中另一半的不愉快，也没有了。

每个人身上，都拖带着他自己的那个世界，由他所见过的、爱的一切所组成的世界。 我们都无法彻底了解和感受他的那个世界。

如果他的世界强烈冲撞到了大多数人的世界，他自己会是最大的受害者，为他惋惜，而不是对他愤怒。反过来，如果大家都搞不定他时，而你却能搞定他，你就是最大的受益者。
